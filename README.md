# wordpress 化した点

- トップ
  - ニュースリスト
  - 背景画像の url
- news
  - ニュースを「投稿」から引っ張ってくる
  - 日付部分は投稿日を表示
  - カテゴリは投稿に設定したカテゴリを表示
  - タイトルが 20 文字以上の記事は、「...」で省略するようにした
  - ページネーション機能を追加(１ページ 15 件まで表示)
- service
  - 画像の url
- works
  - カスタム投稿タイプ「works」を作成
  - works のアーカイブページを表示させている
  - カスタムフィールドで、画像と企業名を変更できるようにした
- company
  - 画像の url
- contact
  - contact form7 を使用して作成。必須項目を入力しないと送信できない。
- news の詳細ページ
  - WEB デザインニュースサイト「ウェブマガジン」に取材いただきました の投稿のみ、中身を作った
  - 画像・画像下のテキスト・大見出し・大見出し下テキスト・小見出し・小見出し下テキスト をカスタムフィールド化
  - 次の記事・前の記事があるとき、ページ下部にリンクが出るようにした
- 全体
  - ヘッダー・フッター・フッター下のページトップへのボタンをパーツ化
  - パンくずリストを function.php に記載し、呼び出せるようにした。page, archive, single で処理を出し分けている。
